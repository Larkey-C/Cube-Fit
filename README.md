# キューブフィット (CubeFit)

1×1×2の直方体を転がし、指定された穴に起立させてスコアを稼ぐエンドレス・パズルゲーム。

## ゲームの概要

- **盤面**: 3×3のグリッド（1-9の数字で表示）
- **直方体**: 1×1×2のサイズで、立っている状態（1マス占有）と寝ている状態（2マス占有）がある
- **目的**: 直方体を穴の位置に起立させてフィットさせる
- **エンドレス**: クリアするたびに新しい穴と直方体が配置される

## 遊び方

### 操作方法

| キー | アクション |
|------|----------|
| 矢印キー（↑↓←→） | 直方体を転がす |
| Space | 決定 |
| Esc | メニュー開閉 |

### 基本ルール

1. **直方体の状態**
   - **立っている**: 1マスを占有
   - **寝ている**: 2マスを占有（水平・垂直）

2. **転がりルール**
   - 立っている状態から移動すると、その方向に倒れて寝た状態になる
   - 寝ている状態から長辺方向に移動させると起立する
   - 寝ている状態から短辺方向に移動させると寝たまま平行移動する

3. **はみ出し許容**
   - 1マス分が盤内にあり、もう1マス分が盤外にはみ出している状態は有効
   - 全てのマスが盤外に出る移動は禁止（エラーアニメーションが再生される）

4. **クリア条件**
   - 直方体が起立した状態で、穴の座標と完全に一致するとクリア
   - スコアが+1され、新しい穴と直方体がランダムに配置される

### ブラインドモード

メニューから「Toggle Mode」を選択すると、視覚的な盤面が非表示になり、テキスト情報だけでプレイできます。

- **表示内容**:
  - 現在の穴の位置（1-9の数字）
  - 直方体の位置と状態（立っている/寝ている）
  - 現在のスコア

## ファイル構成

```
cubefit/
├── index.html          # メインHTML
├── styles.css          # スタイルシート
├── README.md           # このファイル
├── 要件定義書.txt       # 要件定義書
└── js/
    ├── constants.js    # 定数定義
    ├── board.js        # Board クラス（盤面管理）
    ├── cube.js         # Cube クラス（直方体の状態管理）
    ├── renderer.js     # Renderer クラス（描画・アニメーション）
    └── game.js         # Game クラス（メインループ・イベント処理）
```

## 技術スタック

- **描画**: HTML5 Canvas 2D API
- **実装**: Vanilla JavaScript（外部ライブラリ不使用）
- **描画法**: アイソメトリック（等角投影法）による立体的な表現
- **デザイン**: 白・黒・グレー基調のミニマルUI

## クラス設計

### Board クラス
盤面の管理、穴の位置、境界判定を担当。

### Cube クラス
直方体の状態管理、転がり判定を担当。
- 3つの状態: `standing`, `lying-horizontal`, `lying-vertical`
- 状態遷移ロジックによる転がり判定

### Renderer クラス
Canvas描画、アイソメトリック座標変換、アニメーションを担当。
- 立体的な直方体の描画（上面・側面の色分け）
- エラーアニメーション（傾いてからスナップ戻る）

### Game クラス
メインループ、イベント処理、モード切替を担当。
- 通常モードとブラインドモードの切り替え
- スコア管理
- クリア判定とループ処理

## 起動方法

1. ブラウザで `index.html` を開く
2. 「Start Game」ボタンをクリック
3. 矢印キーで直方体を操作して遊ぶ

## 動作環境

- 最新のブラウザ（Chrome, Firefox, Safari, Edge）
- HTML5 Canvas 2D APIをサポートしていること
